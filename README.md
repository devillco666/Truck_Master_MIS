# proyecto Ioan Sorin Muntean
#TRUCK MASTER MIS
Sistema de Seguimiento de Flotas, Gastos y Trabajadores
Funcionalidades Clave
Gestión de la Flota

Registro detallado de cada vehículo (modelo, matrícula, estado).
Mantenimiento programado y histórico de reparaciones.
Seguimiento de ubicación en tiempo real (si se integra con un sistema GPS).
Control de Gastos

Registro de gastos de combustible, reparaciones, y otros costos operativos.
Generación de reportes mensuales y anuales para análisis financiero.
Gráficas y estadísticas para visualización de datos.
Gestión de Trabajadores

Registro de conductores con información de contacto y detalles laborales.
Horarios de trabajo y turnos.
Calificación y evaluación del rendimiento de los conductores.
